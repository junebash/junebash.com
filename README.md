# JuneBash.com

My personal website, built with [Zola](https://www.getzola.org) and the [Apollo](https://github.com/not-matthias/apollo) theme.

## What's Here

- **Code** - Programming projects and iOS apps
- **Music** - Compositions for bands, solo work, and concert pieces
- **Posts** - Writing on programming, philosophy, games, films, and life
- **Now** - What I'm up to lately

## Development

```sh
zola serve   # Local dev server
zola build   # Build for production
```

Visit the live site at [junebash.com](https://junebash.com)
